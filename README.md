# Introduction

## What's this?

`DaVinci` is a more modern CoreGraphics wrapper for iOS/OS X.

## Requirements

* iOS 10.0+
* macOS 10.10+
* Xcode 9 with Swift 4

## Installation

#### CocoaPods

```ruby
pod 'DaVinci'
```
